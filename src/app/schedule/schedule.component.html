    Python é uma linguagem de programação amigável e direta. Quem aprende Python costuma ter domínio sobre as construções básicas da linguagem mas acaba não conhecendo um outro lado dessa linguagem. Existem construções, bibliotecas, utilitários e até comandos básicos que, se usados corretamente, tornam o código Python mais legível, direto e simples. Nesta apresentação discutiremos sobre o jeito Python de escrever código, alguns recursos como list comprehensions, generators e iterators, bibliotecas de manipulação de funções, utilitários para manipular string de forma direta, e outras funcionalidades para se tornar um Pythonista de fato.
